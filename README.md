# E-Commerce Website (Web Development Project)

## Project Description

This project is a simple **E-Commerce Website** developed using **HTML, CSS, and JavaScript**. The purpose of this project is to demonstrate the basic functionality of an online shopping platform where users can view products and add them to a shopping cart.

The website displays a list of products with their names and prices. Users can click the **Add to Cart** button to add selected products to the cart. The cart dynamically updates and displays the items that the user has selected.

This project is designed as a beginner-friendly web development project and can be expanded into a full-stack application by integrating backend technologies and databases.

---

## Technologies Used

* **HTML** – Used to create the structure of the web pages.
* **CSS** – Used to style and design the user interface.
* **JavaScript** – Used to implement interactive features like adding products to the cart.

---

## Project Features

* Display multiple products
* Show product name and price
* Add products to a shopping cart
* Dynamically update the cart using JavaScript
* Simple and responsive layout

## How the Project Works

1. The **HTML file** creates the layout of the website and displays the products.
2. The **CSS file** is used to design the product cards and overall page appearance.
3. The **JavaScript file** handles the cart functionality.
4. When a user clicks **Add to Cart**, the product information is stored in an array.
5. The cart section updates automatically to show the selected items.
